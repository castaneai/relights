## Usage

1. Paste image into window using `Cmd+V` (`Ctrl+V` on Windows)
2. Drag the image upwards to adjust the scale
3. The resized image will be copied to the clipboard!

## Development

```
yarn build && yarn start
```

## Build binary

```
yarn dist
```
